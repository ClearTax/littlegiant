Control the font size of an element using the `lil-text-{size}` utilities.

```html live
<>
  <div>
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
    <h4>Heading 4</h4>
    <h5>Heading 5</h5>
    <h6>Heading 6</h6>
  </div>
  <p class='lil-text-s-12'>text 12</p>
  <p class='lil-text-s-14'>text 14</p>
  <p class='lil-text-base'>text base</p>
  <p class='lil-text-s-18'>text 18</p>
  <p class='lil-text-s-20'>text 20</p>
  <p class='lil-text-s-24'>text 24</p>
  <p class='lil-text-s-32'>text 32</p>
  <p class='lil-text-s-36'>text 36</p>
  <p class='lil-text-s-40'>text 40</p>
  <p class='lil-text-s-48'>text 48</p>
  <p class='lil-text-s-56'>text 56</p>
  <p class='lil-text-s-64'>text 64</p>
</>
```
